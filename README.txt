# Reimbursement Form Application

This is a web application to process reimbursement forms. Users can capture receipts using their phone camera, and the app will extract relevant details and store them in a Google Sheet.

## Features
- Capture receipt images using the phone camera.
- Extract OR number, date, time, and amount from the receipt using OCR.
- Store extracted details in a Google Sheet.
- Add user details (name, ID number, position, division, team head) and month for the reimbursement.

## Setup

### Prerequisites

- Flask
- OpenCV.js
- Google API client
